# HLURLImageView
Obtain and display the image from the server by asynchronous communication

### How to use
```
    @IBOutlet weak var urlImageView: HLURLImageView!
    
    override func viewDidLoad() {
        super.viewDidLoad()

        urlImageView.loadImage(urlString: "https://~~~")
    }
```
 
### Supported
Swift 4.x
